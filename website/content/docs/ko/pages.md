Argus는 사이드바 내비게이션에서 접근할 수 있는 8개의 주요 페이지로 구성됩니다.

## Overview

오늘의 요약을 보여주는 메인 대시보드입니다.

![Overview](/screenshots/overview.png)

## Sessions

상세 이벤트 타임라인과 함께 개별 코딩 세션을 탐색하고 분석합니다.

![Sessions](/screenshots/sessions.png)

## Usage

비용, 토큰, 모델, 효율성, 설정 영향 분석을 5개 탭으로 제공합니다.

![Usage](/screenshots/usage.png)

## Tools

도구 사용 통계, 카테고리 분포, 일별 추이를 표시합니다.

![Tools](/screenshots/tools.png)

## Projects

프로젝트별 비용 비교와 상세 분석을 제공합니다.

![Projects](/screenshots/projects.png)

## Insights

AI 기반 제안, 고비용 세션 분석, 예산 추적을 제공합니다.

![Insights](/screenshots/insights.png)

## Rules

에이전트 설정 파일(CLAUDE.md, settings.json 등)을 보고 편집합니다.

## Settings

테마, 언어, 자동 새로고침, 에이전트 한도, 가격 동기화, 설정 가이드를 제공합니다.
